# Contributing to Kabir Ke Dohe API

Thank you for your interest in contributing to the Kabir Ke Dohe API project! We welcome contributions from everyone.

## How You Can Help

We need your help to make this project better! Here are some ways you can contribute:

- **Fix typos or grammatical errors** in the dohe (couplets) or their translations
- **Improve translations** of existing content
- **Suggest ideas** to enhance our data quality and comprehensiveness
- **Add new dohe** that aren't yet in our collection
- **Improve documentation** to make the project more accessible
- **Fix bugs** or implement new features

## Getting Started

1. Fork the repository
2. Clone your fork to your local machine
3. Create a new branch for your changes
4. Make your changes
5. Submit a pull request

## Guidelines

- Please ensure your code follows the project's coding style
- Include clear commit messages that explain your changes
- Update documentation if necessary
- Add tests for new features when applicable

## Need Help?

If you have any questions or need assistance, please open an issue on GitHub or reach out to the maintainers.
